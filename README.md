## Machine Learning

### Overview
A collection of notebooks covering core Machine Learning concepts.
Each notebook focuses on a specific idea and explores its assumptions, intuition, and working in a clear and structured way, supported by simple visualizations to make the behavior easier to see and understand



### Approach
Learning from first principles with focus on underlying mathematics 



### Purpose
With this kind of understanding, Machine Learning becomes more than just a tool - it becomes a way to solve real-world problems thoughtfully, create meaningful solutions, and build things that truly matter
